## About

This is an alternate frontend for [https://app.ens.domains](ENS), using [wagmi](https://wagmi.sh/), [Rainbow Kit](https://www.rainbowkit.com/) and [Fleek](https://fleek.co/) for hosting.

## Run

```bash
yarn start
```

## Features

- [x] Commit registration
- [x] Register
- [x] Set primary name
- [ ] Set avatar
- [ ] Set additional data
- [ ] Renew
